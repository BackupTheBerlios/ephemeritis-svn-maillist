+ <h2>
+  About Ephemeritis
+ </h2>
+ 
+ <p>
+  <strong>Ephemeritis</strong> is an effort to create an accessible open
+  source ephemeris library that can be available for use by Open Source
+  applications.  It's not yet in a fully working state, but if you have
+  an interest in this topic, you might be keen to assist with its
+  development.  Developers are welcome in the project, particularly if
+  you're skilled at ephemeris calculations already.
+ </p>
+
+ <p>
+  The initial goal is to write/adapt (and test) all the relevant code for
+  the basic problems such as nutation, precession, conversion of units,
+  various different methods of calculating positions, etc.  Having done
+  this, the focus will move towards developing a higher level API for
+  using the lower level code.  When the API is stable, some efforts might
+  be made to write bindings for other languages.
+ </p>
+
+ <h2>
+  Getting Ephemeritis
+ </h2>
+
+ <p>
+  You can find various relevant links and downloads by visiting the <a
+  href="http://developer.berlios.de/projects/ephemeritis/">project page</a>. 
+  There's Subversion repository if you'd like to look
+  at it from a developer's point of view, or alternatively you might simply
+  download the source package.  Please also join the mailing lists and make
+  yourself known -- it's more fun to work on coding these things when there
+  are people taking an active interest.
+ </p>
+
+ <h2>
+  Implementation of Ephemeritis
+ </h2>
+ 
+ <p>
+  Ephemeritis is written in C, and originally based on the GPL'd C++ code of
+  <a href="http://astrolabe.sourceforge.net">Astrolabe</a>, by Bill McClain.
+  Astrolabe implemented many ephemeris routines that were published by
+  Jean Meeus in <em>Astronomical Algorithms</em> (1998), and Ephemeritis
+  development is presently continuing to use this book as a source of
+  information.  (Hey, it's what I have on the shelf, plus it's quite an
+  amazing book.)
+ </p>
+
+ <p>
+  In the future, it will hopefully be completely feasible to include
+  bindings for various other languages, though the primary aim is to get the
+  C library working and develop a stable API that makes sense.
+ </p>
+ 
+ <p>
+  One intent, of course, is for Ephemeritis to be as portable as possible
+  (within reason).  It's packaged using the GNU Autotools, and so far it's
+  known to compile on Debian Linux, NetBSD, and in Sun's Solaris OS.  I've
+  also seen it compile successfully in Windows within a Cygwin environment.
+  I can't vouch for the correctness of the algorithms on any of these
+  platforms, because so far it's had limited testing.
+ </p>
+
